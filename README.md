#知乎日报 Alfred Workflow

###1.安装

* bin 目录为 workflow 包，双击打开即可安装
* src 目录为源码

###2.使用

* 默认使用 `zd` (zhihu daily) 拉取最新内容列表
* 回车打开指定内容页面
